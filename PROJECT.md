# Кулинарная библиотека — инструкция для Claude

## Что это
Личная кулинарная библиотека Юлии. 145 рецептов: PDF-сборники от PraCooking + личные рецепты.

## Структура папки
```
recipes/
  cookbook.html   ← главный каталог, открывать в браузере
  catalog.json                     ← база данных каталога
  update-catalog.py              ← запускать после добавления рецептов
  generate_recipe_html.py          ← генерирует HTML из .txt рецептов
  PROJECT.md                        ← этот файл
  README.md                        ← инструкция для Юлии

  my-recipes/                     ← личные рецепты Юлии
    *.txt                          ← исходники (имена латиницей, kebab-case)
    *.html                         ← красивые HTML-страницы (генерируются автоматически)
    photo/                          ← фото блюд (имя = латинское имя рецепта + .jpg)

  PraCooking/                      ← карточки по продуктам
  PraCooking-BASE-COURSE/          ← базовый курс (большие сборники)
  RECEPIES_PraCooking/             ← сборники рецептов по темам
  photo_food/                      ← фото еды Юлии
```

## Как добавить новый рецепт (через чат)
1. Юлия присылает текст рецепта в чат (любой формат, любой язык)
2. Claude создаёт `.txt` файл в `my-recipes/` с именем латиницей (kebab-case)
3. Claude запускает `update-catalog.py` — генерируется HTML и обновляется каталог
4. Фото: Юлия кладёт в `my-recipes/photo/` с именем как у .txt файла (например `oladushki.jpg`)

## Формат .txt файла рецепта
```
Название: Название блюда (на языке оригинала)
Категория: категория, подкатегория
Теги: #тег1 #тег2
Оценка: хороший ✓  (или: надо попробовать)
Источник: откуда рецепт (опционально)

Ингредиенты:
количество ингредиент
...

Приготовление:
1. Шаг первый
2. Шаг второй
...

Заметки:
- личные комментарии
```

## Важные правила
- Имена файлов — латиница, kebab-case: `zalivnoy-pirog-s-kapustoy.txt`
- Писать **е** вместо **ё** во всех русских текстах
- Не переводить рецепты на другой язык
- Фото называть как .txt файл: `oladushki.jpg` → рецепт `oladushki.txt`
- Скрипт `update-catalog.py` автоматически транслитерирует имена если нужно

## Технические детали
- HTML каталог: поиск по ингредиентам, фильтр по типу, вкладка "my-recipes"
- При клике на личный рецепт открывается красивая HTML-страница с разделами
- PDF-рецепты открываются напрямую с нужной страницей (#page=N)
- Обновление каталога: `cd recipes && python3 update-catalog.py`
